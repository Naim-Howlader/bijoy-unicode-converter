# Bijoy Unicode Converter

Convert Bangla text from Bijoy encoding to Unicode (UTF-8) in JavaScript.

## Installation

### 1. Run this command

```bash
npm install bijoy-unicode-converter
```

### 2. And just use it anywhere you want

